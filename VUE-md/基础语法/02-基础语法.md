 **data** 和 **methods**中的属性和方法都会被挂载到Vue的实列上，在new Vue() 中可以用this.xxx 来调用、访问

```js

    const App = new Vue({
        el:'#app',
        data:{
            counter:0
        },
        methods:{ // 方法
            add(){
                this.counter++
            },
            sub(){
                this.counter--
            }
        }
    })
```
